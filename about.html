Frequently Asked Questions

How did you come up with "Arthropunk"?

When I was younger, I collected insects and post-punk records. Then one day, I was bitten by a radioactive comic book and...Arthropunk

How do you draw your comics?

I sketch the comics with pencil on paper, color with watercolor pencils, and trace over with pen. If you look closely, you might be able to spot the points where one pen started dying and I, in denial, refused to replace it.

How did you build your website?

I learned how to code HTML and CSS and wrote the crappy code for the site myself. If you're building a website (or if you're an actual coder and see something that needs fixing) you can take a look at all my code on Github.
